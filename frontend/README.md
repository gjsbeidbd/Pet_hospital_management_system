# 宠物医院信息管理系统 - 前端

本系统是基于Vue 3和Element Plus构建的宠物医院信息管理系统前端部分。

## 技术栈

- Vue 3
- Element Plus
- Vue Router
- Axios

## 推荐的IDE设置

[IntelliJ IDEA](https://www.jetbrains.com/idea/) 或 [VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) 插件。

## 浏览器支持

- 现代浏览器（Chrome, Edge, Firefox等）

## 项目设置

```sh
npm install
```

### 编译和热重载（开发模式）

```sh
npm run dev
```

### 编译和压缩（生产模式）

```sh
npm run build
```

### 系统特性

- 多角色访问控制（宠物主人、前台、医生、院长）
- 用户认证和权限管理
- 宠物信息管理
- 在线预约挂号
- 电子病历记录
- 费用管理
